# 📋 Project Overview

## What is BoniKeytype?

BoniKeytype is a web-based typing practice platform designed to improve typing speed and accuracy. Users practice across multiple modes, track their progress over time, compete with others on leaderboards, and race in real-time multiplayer sessions.

---

## 🎯 Goals

- Provide a distraction-free, responsive typing environment
- Support multiple practice modes for beginners and advanced typists
- Track and visualize progress with detailed statistics
- Encourage consistency through gamification and daily challenges
- Enable real-time competition with friends or global users

---

## 👤 Target Users

| User Type | Use Case |
|---|---|
| Students | Improving academic typing speed |
| Professionals | Practicing touch typing for productivity |
| Competitive typists | Training for speed records and leaderboard rankings |
| Casual users | Daily warm-up typing sessions |

---

## ✨ Feature Summary

### Typing Modes
- **Timer Mode** — Type as many words as possible within a time limit (15s, 30s, 60s, 120s)
- **Word Count Mode** — Complete a fixed number of words as fast as possible
- **Quote Mode** — Type real quotes sourced from literature or the web
- **Custom Text Mode** — Paste or type your own text for practice

### Statistics Tracked
- Words per minute (WPM)
- Accuracy percentage
- Error count
- Character count (correct / incorrect / extra / missed)
- Progress history over time

### User Accounts
- Register and login
- Password management
- Personal stats dashboard
- Historical test results

### Leaderboards
- Global all-time rankings
- Daily leaderboard resets
- Friends-only leaderboard

### UI Customization
- Dark / light mode toggle
- Font selection (monospace, serif, sans-serif)
- Caret styles (block, underline, bar)
- Color themes

### Multiplayer
- Real-time typing races
- Create or join a race room
- Live progress bars for all participants
- Post-race stats comparison

### Gamification *(Optional)*
- XP points awarded per test
- Levels based on cumulative XP
- Achievement badges (e.g., "100 WPM Club", "7-Day Streak")
