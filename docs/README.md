# ⌨️ BoniKeytype

> A web-based typing practice platform to improve speed, accuracy, and consistency.

BoniKeytype lets users practice typing across multiple modes, track personal stats, compete on leaderboards, and race others in real time.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/your-username/bonikeytype.git
cd bonikeytype

# Start frontend
cd frontend && npm install && npm run dev

# Start backend
cd backend && npm install && npm run dev
```

See [docs/setup.md](docs/setup.md) for full environment configuration.

## 🏗️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + TailwindCSS |
| Backend | Node.js + Express |
| Database | PostgreSQL |
| Real-Time | Socket.io |
| Auth | JWT + bcrypt |

---

## ✨ Key Features

- **Typing Modes** — Timer, word-count, quote, and custom text modes
- **Live Stats** — WPM, accuracy, error count, character tracking
- **User Accounts** — Profiles, history, and progress tracking
- **Leaderboards** — Global, daily, and friends rankings
- **UI Customization** — Dark/light mode, fonts, caret styles, themes
- **Multiplayer** — Real-time typing races via WebSocket
- **Gamification** — XP, badges, and levels *(optional)*
